# Vietnam Textbook Covers, 1980-2026

Working collection for finding cover images of Vietnamese school textbooks published or used in Vietnam from 1980 through 2026, for an art/research project.

## Current Contents

- `images/`: downloaded PNG/JPEG cover images.
- `metadata.csv`: one row per image, with title, grade, subject, era/date confidence, local filename, source image URL, page URL, and rights notes.
- `sources/`: saved HTML copies of source pages used in this pass.
